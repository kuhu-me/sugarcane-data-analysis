## Project Overview

This project focuses on analyzing a global **sugarcane production dataset** containing country-wise agricultural statistics. The primary goal is to clean, standardize, and prepare the data for meaningful analysis and future modeling.

At this stage, the project emphasizes **handling real-world data issues**, such as inconsistent numeric formats and data type conversion.

---

## Dataset Description

The dataset contains country-level sugarcane production data, including:

- Country and continent information
- Total sugarcane production
- Production per person
- Cultivated land area
- Yield per hectare

The raw data uses **European-style number formatting**, which required careful preprocessing.

---

## Work Completed So Far ✅

### Data Preprocessing & Cleaning
- Converted object-type numeric columns to proper numeric formats
- Handled thousands separators (`.`) and decimal commas (`,`)
- Standardized columns such as:
  - Production (Tons)
  - Production per Person (Kg)
  - Yield (Kg / Hectare)
- Managed invalid and missing values safely
- Ensured consistency across all numeric columns

### Data Handling
- Corrected data types for analysis readiness
- Prepared clean, structured data suitable for visualization and modeling
